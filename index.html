Cub3d

Cub3D

This project is a ray-caster implementation using the minilibX library.
It aims to recreate the immersive experience of exploring a maze in a first-person perspective.

Features:

  • Wall collision
  • Texture mapping
  • Sprite rendering
  • Player movement
  • Shoot system
  • Minimap

This project is interesting to me because it's my first video games, He taught me the raycasting and the texture mapping, which are required tasks.
However, I chose to include additional features, including a basic enemy system and a shooter system.
